## Mood detector app

**Topic** : Facial emotion recognition

**Platform** : Python3.6

## To run:

Syntax:
```
python aithon_level2.py <train_file_path> <test_file_path>
```

Example:
```
python aithon_level2.py data/aithon2020_level2_traning.csv data/aithon2020_level2_testing.csv
```
