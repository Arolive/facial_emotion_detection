## Mood detector app

**Topic** : Facial emotion recognition

**Platform** : Python3.6

## To run:

Call the api using:
```
import aithon_level2 as api

```

## Directory layout

- source     : contains the main `.py` file for the classifier
- notebook   : contains `.ipynb` notebooks for practice
- data       : contains the dataset
